# Developing This App

## Downloading
Clone the repo

```
$ git clone git@github.com:The-Politico/politico-interactive-templates.git
```

## Developing
Run the development watch script:

```
$ yarn dev
```

Then you can edit files in the `src/` directory and see them automatically build.

## Publising
Run tests:

```
$ yarn test
```

Ship it:
```
npm version patch
npm publish
```
